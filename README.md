### 🛸 4. SEDS Attendance Portal — Internal Mission Control

<div align="center">

> A **fully serverless, role-based PWA** for managing KCT SEDS club internals — attendance, meetings, automated emails & analytics. All powered by Google Apps Script + Sheets. Live at `attendance.kumaraguruseds.space`.

<details>
<summary><b>📖 Full Project Details (click to expand)</b></summary>

#### 🔐 3-Role Authentication System
| Role | Dashboard |
|------|-----------|
| `Member` | Personal attendance %, per-team stats, weekly meeting timetable |
| `Lead` | Mark attendance, schedule meetings, view session history |
| `Admin Pro` | Full cross-team table with search & filter by name, team, date |

#### 🛡️ Automated 3-Strike System
```
Present              → strikes = 0  (full reset)
Absent (no reason)   → strikes + 1  (always)
Absent (1st reason)  → excused      (no strike)
Absent (2nd+ reason) → strikes + 1  (abuse detected)
Strikes ≥ 3          → termination email auto-fired
```

#### 📧 3-Layer Email Failover
```
Main Gmail → Worker 1 (UrlFetchApp) → Worker 2
```
Triggers: ✅ Present · 🟡 Excused · 🟠 Warning · 🔴 Terminated · 📩 Meeting Invite · 🚫 Cancellation

#### 🌠 Visual Design
- Space-themed background with **200 animated falling stars**
- **Glassmorphism cards** with `rgba` backgrounds and subtle borders
- Fully responsive — 4-col → 2-col grid on mobile (`≤600px`)
- **Poppins** font, custom SVG login icons

#### ⚙️ Apps Script Backend (4 Files)
| File | Role |
|------|------|
| `Attendance.gs` | Login, members, attendance submit, stats |
| `Meeting_Scheduler.gs` | Schedule, fetch, cancel meetings + conflict detection |
| `Mailer.gs` | Invite/cancel emails + admin master report |
